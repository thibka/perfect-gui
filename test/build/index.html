Perfect GUI API

Basics

const gui_1 = new perfectGUI({
    name: 'Basics',
});

gui_1.addButton('Button', () => {
    element.style.backgroundColor = getRandomColor();
    element.style.backgroundImage = 'none';
});

gui_1.addSlider('Slider', 
    { min: 0, max: 2, value: 1, step: .01 }, 
    value => element.style.transform = `scale(${ value })`
);

gui_1.addSwitch('Switch', true, state => {
    if ( state ) element.classList.remove('round');
    else element.classList.add('round');
});

gui_1.addList('List', ['red', 'pink', 'yellow', 'blue'], item => {
    element.style.backgroundColor = item;
    element.style.backgroundImage = 'none';
});

gui_1.addImage('Image 1', 'path/to/image-1.jpg',
    path => {
        element.style.backgroundImage = `url(${path})`;
        document.querySelector('#container-1 .note').textContent = "Photo by Joel Filipe on Unsplash";
    }
);

gui_1.addImage('Image 2', 'path/to/image-2.jpg',
    path => {
        element.style.backgroundImage = `url(${path})`;
        document.querySelector('#container-1 .note').textContent = "Photo by Milad Fakurian on Unsplash";
    }
);

Positioning

GUI instances can be positioned in any corner of the screen / container.

When multiple instances share the same position (like GUI 1 and GUI 2 in the example below), they are stacked next to each other.

const gui_1 = new perfectGUI({
    name: 'GUI 1',
    width: 200
});

gui_1.addButton('Buttons can handle multiple lines of text.', () => {
    element.style.backgroundColor = getRandomColor();
});

const gui_2 = new perfectGUI({
    name: 'GUI 2',
    width: 200
});

gui_2.addButton('Button', () => element.style.backgroundColor = getRandomColor() );

const gui_3 = new perfectGUI({
    name: 'GUI 3',
    position: 'top right'
});

gui_3.addButton('Button', () => element.style.backgroundColor = getRandomColor() );

const gui_4 = new perfectGUI({
    name: 'GUI 4',
    position: 'right bottom'
});

gui_4.addButton('Button', () => element.style.backgroundColor = getRandomColor() );

Folders

Folders can be closed by default by setting the open parameter to false. 
const gui = new perfectGUI({
    name: 'Folders'
});

let folder_1 = gui.addFolder('Folder 1');

folder_1.addButton("Random color", () => {
    element.style.backgroundColor = getRandomColor();
});

let folder_2 = gui.addFolder('Folder 2', false);

folder_2.addButton("Random color", () => {
    element.style.backgroundColor = getRandomColor();
});

Other options

GUI panels can be dragged around with the draggable option.

They can also have a custom width, by using the width option.

The toggleClose() method can be used to open / close a GUI panel.

Just like folders, GUI panels can be closed by default by setting the close option to true.

const gui_1 = new perfectGUI({
    name: 'GUI 1 (drag me!)',
    width: 500,
    draggable: true
});
gui_1.addButton('Custom width using the `width` option', () => {});

const gui_2 = new perfectGUI({
    name: 'GUI 2 (drag me!)',
    close: true,
    draggable: true,
    position: 'bottom left'
});

gui_2.addButton('gui_1.toggleClose();', () => {
    gui_1.toggleClose();
});

const gui_3 = new perfectGUI({
    name: 'GUI 3 (closed)',
    closed: true,
});

gui_3.addButton('gui_2.toggleClose();', () => {
    gui_2.toggleClose();
});
